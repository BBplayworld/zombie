# 🚀 빠른 시작 가이드

## 1단계: 에셋 생성

게임 에셋이 없으므로 먼저 테스트용 에셋을 생성합니다.

```bash
# 브라우저에서 에셋 생성기 열기
start tools/generate-assets.html  # Windows
open tools/generate-assets.html   # macOS
xdg-open tools/generate-assets.html  # Linux
```

또는 직접 파일을 브라우저로 드래그하여 엽니다.

"모든 에셋 생성 및 다운로드" 버튼을 클릭하면:
- `bg1.png` (배경 타일 1)
- `bg2.png` (배경 타일 2)  
- `player.png` (플레이어 스프라이트)

다운로드된 파일들을 `public/zombie/assets/` 폴더에 저장합니다.

## 2단계: 의존성 설치

```bash
npm install
```

## 3단계: 개발 서버 실행

```bash
npm run dev
```

브라우저에서 자동으로 열리거나 `http://localhost:3000` 접속

## 🎮 게임 시작!

1. 로딩 완료 후 "게임 시작" 버튼 클릭
2. WASD 또는 방향키로 캐릭터 이동
3. 배경이 자연스럽게 무한으로 연결되는 것을 확인
4. ESC 키로 일시정지/재개

## ⚠️ 문제 해결

### 이미지가 로드되지 않는 경우
- `public/zombie/assets/` 폴더에 이미지 파일이 있는지 확인
- 파일 이름이 정확한지 확인: `bg1.png`, `bg2.png`, `player.png`
- 브라우저 콘솔(F12)에서 에러 확인

### 게임이 시작되지 않는 경우
- 브라우저 콘솔에서 에러 확인
- Nuxt 개발 서버가 정상 실행 중인지 확인
- 포트 3000이 다른 프로그램에 의해 사용 중이지 않은지 확인

### 성능 이슈
- 브라우저 하드웨어 가속 활성화 확인
- 다른 탭/프로그램 닫기
- FPS 표시로 성능 모니터링
